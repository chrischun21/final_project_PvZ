# INSTRUCTIONS FOR GAMEPLAY

open the .sln file and click 'Run' to start


Use the mouse to navigate and left-click to select options
and perform actions.

Press 'M' to mute all sounds.

During the gameplay loop, the ally Plants can be selected by
hitting the corresponding key in the top left of the inventory:
(1) : Peashooter
(2) : Snow Pea
(3) : Repeater
(4) : Plantern
(5) : Wallnut
(6) : Potato Mine
(7) : Squash
The selected Plant can be placed by hovering over a valid 
position and left-clicking.

On the victory/loss screen, you can press
'P' to play again
'L' to return to the menu 



# PROJECT DESCRIPTION
Our group project is a UT-themed 2D tower defense game, inspired by Plants vs. Zombies’ Vasebreaker mode. The game is set in a tile-based system with many vases. Broken vases have a chance to spawn a placeable ally or an enemy. The enemies attempt to move to the base on the left side of the screen, while the allies oppose them and defend their base. Each ally has a different set of abilities and attributes to stop the enemy in various ways. The level is won when all the vases are broken and there are no enemies left on the field. The level is lost if enemies reach the defense target.
